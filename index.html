The Kancolle Database! Translation is a userscript created for Chrome and Firefox that translates Database Kancolle to English for all non-native Japanese speaking audiences. Instead of having to worry about Google Translate or a broken set of Japanese language skills, you can easily navigate, search, and add your own recipes to this awesome database.

Installation (Chrome)

  1. Install Tampermonkey
  2. Click the large install link at the top of this page.
  3. Press OK on in the new Tampermonkey tab.

Installation (Firefox)

  1. Install Greasemonkey Add-on
  2. Click the large install link at the top of this page.
  3. Wait out the timeout and install the script!

Compatibility

This script should have top priority over anything that may modify the page, otherwise you may come up with weird translation errors. But for the most part, it should be fine.

Troubleshooting

If you are having any problems at all, please make sure you are on the latest version of all software (Chrome and Tampermonkey / Firefox and Greasemonkey). This script was tested mainly on Chrome 32 and Firefox 26.0.
